# 任务协作管理系统前端
## 安装

1.  安装node.js

2.  安装npm淘宝镜像

a.  运行cmd

b.  输入： npm install -g cnpm --registry=https://registry.npm.taobao.org

c.  命令行输入  cnpm install(推荐)  或  npm install


3. 配置后端地址

a.vue.config.js    const DEV_URL = 'http://127.0.0.1:8888/'; 配置后端地址

b.run serve 运行服务

c.http://localhost:8045/ 访问应用
